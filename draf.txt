1.	NoSQL (Not Only Structured Query Language)
Menurut Gaurav Vaish berdasarkan Wikipedia (2013:2): 
	In computing, NoSQL (mostly interpreted as “not only SQL”) is a Broad class of database management system identified by its non-adherence to widely used relational database management system model; that is, NoSQL databases are not primarily built on tables, and as a result, generally do not use SQL data manipulation.
Dalam Komputer, NoSQL (lebih sering diartikan sebagai “tidak hanya SQL”)  merupakan suatu bagian dari kelas sitem menejemen  database diidentifikasi dari ketidaksesuaian penggunaan model relasional system manajemen database secara luas, yang berarti NoSQL database tidak mengutamakan dibuat table, dan sebagai hasil, umumnya tidak menggunakan SQL untuk memanipulasi data.
